### Quickstart:

1. Ensure you have installed the following (or install packages via `requirements.txt`):
- Python 3
- Jupyter Notebook
- Pandas
- Numpy

2. In the shell, run Jupyter Notebooks:
```bash
jupyter notebook Your_first_neural_network
```
